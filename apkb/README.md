# APK Builder Bot

Telegram-бот для автоматической сборки Android APK из ZIP-архивов с веб-сайтами.

## Возможности

- Принимает ZIP-архив с HTML/CSS (только текст, без изображений)
- Проверяет безопасность (фильтрация файлов, очистка HTML, запрет внешних ссылок)
- Управляет ресурсами сервера (память, очередь)
- Возвращает подписанный APK
